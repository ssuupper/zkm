# HOW 
### step 1: checkout source ######
```bash
git clone https://github.com/ssuupper/zkm.git
```
### step 2: install dependencies ######
```bash
cd zkm
npm install
npm install electron -g
npm install gulp -g
```
### step 3: gulp ######
```bash
gulp
```
### step 4: start ######
```bash
npm start
```
